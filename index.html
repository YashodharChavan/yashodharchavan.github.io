Yashodhar Chavan Portfolio

Welcome to the web-based portfolio of Yashodhar Chavan, a frontend developer specializing in React and modern web UI. This site is a recreation of macOS X Tiger 10.4, showcasing applications like Safari, Finder, Spotlight, Terminal, Dashboard, and more.

Every component is interactively built using React, CSS, and HTML to mimic the Apple experience in a web browser. You can explore projects, learn more about the developer, and use web-based versions of classic Mac applications.
